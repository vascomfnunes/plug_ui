# PlugUI — Visual System Specification

## DESIGN PRINCIPLES

PlugUI follows these visual philosophy principles:

- **Premium, Restrained Aesthetic:** Subtle micro-gradients, calm charcoal-blue background, and carefully managed opacities.
- **Subtle Gold Accent:** Value arcs and active states use the gold accent tastefully—do not overuse it.
- **Calm Layout:** Elements have breathing room. Focus on alignment and visual balance.
- **Low-Noise:** Textured, but never glaring. TV static or overt textures break the aesthetic.
- **Clean Forms:** Rounded corners, fine outlines, legible kerning. Let standard geometry shine.
